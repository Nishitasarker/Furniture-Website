# 🪑 FURNS - Premium Furniture Store

FURNS is a modern, responsive, and production-ready furniture e-commerce platform. It provides a seamless shopping experience, allowing users to explore high-quality furniture, manage a wishlist/cart, and enjoy a premium interface.

## ✨ Key Features

### 🏠 User Experience
* **Category Highlights:** Dedicated grid layout for Bedroom, Living Room, Dining, and Office furniture.
* **Featured Collection:** Showcase of best-sellers and popular items.
* **Flash Sale & Offers:** Dynamic banners for seasonal discounts.
* **Why Choose Us:** Section highlighting premium quality, warranty, and delivery services.
* **Customer Testimonials:** Slider showcasing real user experiences and feedback.
* **Interior Tips (Blog):** SEO-optimized section for furniture care and interior design advice.
* **Newsletter & Socials:** Subscription portal and integrated social media links.

### ⚙️ Dashboard & Management
* **Admin Dashboard:** A private area to manage inventory and view store metrics.
* **Add/Manage Items:** Full CRUD functionality to expand the FURNS catalog.
* **Cart Management:** Real-time tracking of selected items.

## 📂 Project Architecture

The project follows a clean Full-Stack Next.js structure:

```text
FURNS/
├── app/             # Next.js App Router (Pages & API endpoints)
├── components/      # Reusable UI components (Cards, Navbar, Footer, Forms)
├── context/         # Global state management (Cart, Authentication)
├── lib/             # Utility functions and Database connections
├── public/          # Static assets (Images, Icons)
├── styles/          # Global styles (Tailwind configuration)
└── next.config.ts   # Next.js configuration settings\

 Getting Started
1. Clone the repository:

Bash
git clone [https://github.com/Nishitasarker/Furniture-Website.git](https://github.com/Nishitasarker/Furniture-Website.git)
cd Furniture-Website

2. Install dependencies:

Bash
npm install

3. Run the development server:

Bash
npm run dev

Open http://localhost:3000 to view the project.
